override void Weather_Thunderstorm.ExitWeatherEffect ( GameObject  gameobject)
virtual

How this works per rain type

  1. I activate a timer since I want it to reset and not start again after deactivateing the particlesystem
  2. Turn off emission so the rain stops falling before we turn off the particle
  3. When exit start fading sound out if this is true 3.1. Set this to false again so we can turn on the sound again when this weather effect happens again
  4. When this timer ends turn off the active rain 4.1. Set back start timer to 0.0f 4.2. Deactivate the particle system
